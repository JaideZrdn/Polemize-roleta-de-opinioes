## O que é?  

Entre em um universo de controvérsias, opiniões divididas e debates acalorados com **Polemize**! Você nunca sabe qual pergunta quente vai surgir, Polemize te surpreende com questões provocativas, curiosas ou até ridiculamente engraçadas.

**Polemize** é um app para criar conversas em uma roda de amigos, gerando discussões divertidas e imprevisíveis sobre diversos temas.  Com o subtítulo de **"Roleta Russa das Opiniões"**, o app surgiu de uma discussão em grupo sobre como seria interessante ter uma ferramenta que facilitasse aquela **intriga gostosa** e debates provocativos entre amigos.

O objetivo? Se divertir em grupo, discutindo sobre temas engraçados e/ou sérios, criando momentos memoráveis de risadas e argumentação.

---

## Como Funciona

Reúna seus amigos, insira os nomes deles, escolha os temas que poderão aparecer e deixe o Polemize fazer o resto! 

<img height="1500" alt="Anuncio3" src="https://github.com/user-attachments/assets/43f510c5-c2ec-4da9-b8ee-44765ac607eb" />

### Dois Modos de Jogo

**Modo com IA:**
- O app usa **inteligência artificial** para analisar as características dos jogadores, preferências e relacionamentos
- A IA cria controvérsias personalizadas baseadas no perfil do grupo
- Decide automaticamente quem debate com quem e qual será o próximo tópico controverso
- Torna cada jogo único, imprevisível e ainda mais divertido

<img height="1500" alt="Anuncio4" src="https://github.com/user-attachments/assets/0157d748-5df8-4c27-b6eb-3ae35d858849" />

**Modo sem IA:**
- Seleciona aleatoriamente polêmicas de uma base de tópicos pré-definidos pelo usuário
- Debates divertidos e provocativos 
- Diversão garantida de forma mais tradicional

<img height="1500" alt="Anuncio5" src="https://github.com/user-attachments/assets/b6226c9d-7d6e-4a0b-a89e-3dff5b3a9273" />

## Minha Participação

### Desenvolvimento Técnico

**Sistema de Alimentação de Dados:**
- Implementação do sistema de carregamento e gerenciamento das polêmicas via **JSON**
- Estruturação de dados para suportar múltiplas categorias de temas

**Navegação:**
- Desenvolvimento do sistema de navegação utilizando padrão **Coordinator** com **SwiftUI**

**Arquitetura:**
- Organização do projeto seguindo **Clean Architecture** combinada com padrão **MVVM**
- Separação clara de responsabilidades garantindo manutenibilidade

**Gerenciamento de Estado:**
- Implementação da passagem de informações entre Views utilizando componentes nativos do **SwiftUI** com **ViewModels**
- Sincronização de estado do jogo entre diferentes telas

### Ideação

- Participação ativa na **ideação do projeto**, que surgiu de uma discussão em grupo sobre a necessidade de um app para debates divertidos entre amigos

---

## Tecnologias Utilizadas

- **SwiftUI** - Framework para desenvolvimento da interface do usuário
- **Coordinator Pattern** - Gerenciamento de navegação e fluxo entre telas
- **JSON** - Estrutura de dados para armazenamento e carregamento das polêmicas
- **Clean Architecture + MVVM** - Organização do código com separação clara de camadas
- **Integração com IA** - Sistema de personalização de controvérsias (Modo IA)
